# 빌드 실행 제한 규정 (Build Execution Restriction Rule)

## 개요
본 규정은 사용자의 명시적인 명령이 없는 경우 에이전트가 모바일 앱(Flutter) 빌드 스크립트나 도커 빌드 등의 컴파일 작업을 임의로 실행하지 못하도록 제한합니다.

## 세부 규칙
1. **임의 빌드 금지**: 에이전트는 코드 수정, 종속성 추가, 환경 설정 변경 등의 작업을 마친 후 자동으로 모바일 APK/AAB 빌드(`build_android_apk.sh` 실행 등)나 도커 이미지 빌드 작업을 시작해서는 안 됩니다.
2. **명시적 명령 조건**: 에이전트는 사용자가 한국어로 **"내가 빌드해줘"** (또는 이와 직접적으로 상응하는 빌드 실행 요청)라고 직접 채팅창에 명령한 경우에만 빌드 스크립트 실행 도구를 호출할 수 있습니다.
3. **대안 제안**: 빌드가 필요한 코드 수정을 완료한 경우, 에이전트는 수정된 내용을 설명한 뒤 "빌드를 진행하려면 '내가 빌드해줘'라고 말씀해주세요"라고 사용자에게 친절히 안내해야 합니다.
